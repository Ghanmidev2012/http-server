# http-server
my first http server 

# How it work
For Test you must have this things
* golang 1.26 + or old version
* and IDE like vscode or cmd
# Commands
First to run write this in terminal
```cmd
go run main.go
```
And open browser and type this
```
http://localhost:8080/hello
```
